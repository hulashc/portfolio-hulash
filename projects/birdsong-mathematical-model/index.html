Spatiotemporal Acoustic Manifold

  • →Birdsong treated as a trajectory through 3D space rather than just a waveform.
  • →Each point encodes the timbre as high-dimensional MFCC features.
  • →PCA compresses features into PC1, PC2, PC3 (X, Y, Z axes).
  • →Loops, twists, clusters reveal phrases, motifs, and tonal vs noisy sections.
  • →Inspired by Lucio Arese's "Visualizing Bird Songs: A Mathematical Model".

Technical Implementation

Feature Extraction · Python + librosa

40 MFCCs + chroma + spectral centroid/bandwidth + rolloff + zero-crossing rate per frame. Features standardised and reduced to 3D with PCA. Frame-aligned time and energy exported to JSON.

3D Manifold · Three.js + WebGL

Points smoothed with Catmull–Rom spline to create a continuous ribbon path. Moving dot synced to audio.currentTime → frame index. Dot size and colour map to energy.

Visual Design

Fading comet tail shows last 280 frames. Ghost trajectory reveals global shape at low opacity. Axes labelled PC1 / PC2 / PC3.

Bird Song Mathematical Model

Project Details

A mathematical model visualised as an interactive 3D web experience — exploring the patterns and structures within birdsong compositions. Inspired by Lucio Arese's "Visualizing Bird Songs: A Mathematical Model".
